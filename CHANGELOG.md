# Changelog

All notable Arleco changes are documented here. Version numbers match `VERSION` at the repo root.

## [0.1.0-beta] — 2026-03-20

First beta aimed at invited creators.

### Added
- Creator studio (graph editor, resources, publish / schedule chapters)
- Public reader (Discover, series pages, episode player)
- Conservatory learn mode
- Standalone account page and reader navigation
- Scheduled chapter publishing (`publishAt`)
- GitHub Pages deploy workflow (`docs/DEPLOY.md`)
- Discord community link in help/footer

### Changed
- Operator branding: Arius Interactive (replacing dissolved Arius Black Studios)
- Hero stat: chapters read this week (not unique visitors)
- Footer pages: about, help, guidelines, terms, privacy, contact
