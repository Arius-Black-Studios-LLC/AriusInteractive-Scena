# Arleco blog content plan

Target audience: **visual novel creators** (writers, artists, solo devs, indie studios) — not players.

Goal: become the blog creators find through Google when researching tools, craft, and publishing.

Publish cadence: **2–3 SEO-focused articles per week** when possible.

## On-page SEO checklist (every post)

- [ ] Title + H1 with primary keyword
- [ ] Meta description (150–160 chars)
- [ ] Canonical URL on `https://arleco.app/blog/...`
- [ ] JSON-LD `Article` schema
- [ ] Internal links to 2+ other blog posts + `/studio` or `/learn`
- [ ] CTA box at bottom (white button on red — see `blog.css`)
- [ ] Add to `sitemap.xml` when published
