About Arleco

Arleco is a browser-first home for episodic visual novels — stories you read chapter by chapter, with branching choices, inventory, and creator tools built in.

For readers

Discover free series, pick up where you left off, and follow creators as they publish new chapters. No app install required — everything runs in your browser on desktop or mobile.

For creators

The Creator studio is a graph-based editor for branching stories: cast characters, wire choices, set metrics and key items, preview episodes, and publish when a chapter is ready. The Conservatory offers guided lessons if you are new to interactive fiction.

Operator

Arleco is built and operated by Arius Interactive — an independent project working toward a formal studio. It is not a registered company or LLC yet.
